SD Storage

SD storage is the primary file system for the project, holding books, voice notes, captured OCR text, and firmware-related assets.

SD Use Cases

  • Store text and eBook files for reading
  • Record voice notes to WAV files
  • Cache OCR-scanned text and AI session output
  • Persist configuration, logs, and updates

SD Management

The firmware uses mutex and queue-based synchronization to ensure safe SD card access from audio and UI tasks running on different cores.

Recommended Card

Use a high-quality microSD card rated for embedded applications. Prefer UHS-I or class 10 cards for reliable audio and file operations.
