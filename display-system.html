Display System

The display system uses an SPI-based TFT panel combined with LVGL for user interaction. This page covers display wiring, UI refresh strategy, and theming controls.

Display Components

  • Color TFT display module
  • XPT2046 resistive touch controller
  • SPI interface for display drawing and touch events
  • LVGL-based screen rendering and widgets

UI Themes

The firmware supports multiple display themes and font scaling to provide a comfortable reading experience. Theme options include light, dark, sepia, and high-contrast modes.

Refresh Strategy

LVGL is updated continuously while display drawing is synchronized with the TFT driver. Touch input is polled and dispatched to the UI task for responsive screen navigation.
