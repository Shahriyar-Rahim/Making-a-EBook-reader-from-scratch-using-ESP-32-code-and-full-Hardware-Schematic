Camera System

The camera system leverages the ESP32-S3 CAM module to capture images for OCR scanning and visual data capture. This page covers camera configuration, capture modes, and integration with OCR.

Camera Data Flow

  • Capture image from camera module
  • Store frame temporarily in memory
  • Convert image to OCR-friendly format
  • Send text to AI and note workflows

Capture Modes

Support includes manual capture for single-page scans and continuous capture for quick multi-page workflows. The UI provides controls to trigger, preview, and save camera frames.

Performance

Camera capture is optimized to keep the UI responsive while image processing and OCR tasks run in the background. The architecture minimizes frame latency and memory pressure on the ESP32-S3.
