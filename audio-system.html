Audio System

The audio system includes recording from an INMP441 microphone, WAV file management on SD storage, and playback through a MAX98357A amplifier.

Audio Capabilities

  • Voice note recording to SD card
  • Playback with progress and control UI
  • Cross-core audio task coordination using FreeRTOS
  • Future TTS/STT integration planned

Current Implementation

Existing functionality preserves working audio recording and playback while adding UI state, playback progress, and file management support.

Future Expansion

Future software work will expand audio into a full audiobook mode, support speech synthesis, and add offline transcription options for recorded notes.
