Features

This page documents every feature category in the ESP32-S3 CAM AI eBook Reader, including reader controls, audio, camera, AI, OCR, networking, and system-level capabilities.

Reader Features

  • Text rendering for .txt and EPUB file formats on the SPI TFT display.
  • Font size and theme controls for light, dark, sepia, and OLED modes.
  • Page navigation with progress tracking, bookmarks, and chapter access.
  • AI-powered summarization, note extraction, and study question generation.

AI Features

  • Cloud AI support with Gemini and OpenAI call flows.
  • Book-aware chat using the current reading context.
  • Summaries, MCQs, short questions, viva questions, and note extraction.
  • OCR cleanup for scanned text readability.

OCR Features

The system captures images using the ESP32-S3 CAM and converts them to text that can be reviewed, cleaned, and saved to the SD card.

Camera Features

Integrated camera support enables document capture, page scanning, and vision-based workflow extensions for future AI vision tasks.

Audio Features

  • Voice note recording using INMP441 and I2S to WAV on SD.
  • Playback through MAX98357A speaker output.
  • Voice note library with delete and audio controls.
  • Planned TTS/STT architecture for audiobook and transcription features.

Network Features

  • Wi-Fi connectivity for AI and OTA.
  • AP mode and web dashboard for configuration.
  • OTA updates via SD card or local network upload.

Battery and Power Features

The system is designed to support battery monitoring, charging state detection, and low-power display operation for a portable reading experience.

Drawing and Notes Features

A built-in drawing and note layer simplifies markup and manual annotations intended for digital notebook workflows.
