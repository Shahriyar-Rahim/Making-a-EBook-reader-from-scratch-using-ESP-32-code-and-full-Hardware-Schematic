Software

The firmware is built using PlatformIO with the Arduino framework and LVGL for the UI. The software stack integrates audio, camera, SD card storage, network connectivity, and AI service calls.

Firmware Stack

  • PlatformIO build system
  • ESP32-S3 Arduino framework
  • LVGL 8.x for UI rendering
  • TFT_eSPI display driver

Subsystem Modules

  • `audio_engine.cpp/h` — I2S playback, recording, and voice note management
  • `book_engine.cpp/h` — Text and page navigation APIs
  • `reader_ui.cpp/h` — Main user interface, controls, and widgets
  • `network.cpp/h` — Wi-Fi and HTTP request handling

Key Software Features

  1. Concurrent UI rendering and audio processing using FreeRTOS.
  2. SD card file access with mutex locking for safe cross-core operations.
  3. AI workflow support for summarization, notes, and OCR text enhancement.
  4. OTA update support and network-based logging.

Development Notes

Recommended workflow: edit source files in `src/`, build with PlatformIO, and deploy via USB with `pio run --target upload`. Maintain the UI state in `reader_ui.cpp` and keep audio control logic centralized in `audio_engine.cpp`.
